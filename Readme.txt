He empezado las practicas en el departamento de informatica de una empresa llamada TecnoRED SL, 

y mi mision es que organizar y documentar varios equipos y la red interna para tener todo el trabajo centralizado.
¿Por qué la empresa usa Git y no carpetas normales?
Para que cada actualizacion del archivo se registra un cambio y volver a la version anterior.
¿Que pasaria si alguien borra un archivo por error?
Que se podria recuperar si hubo un commit.
